- Gmail --> [ProtonMail](https://protonmail.com/)
- Google Search --> [Duck Duck Go](https://duckduckgo.com/)
- Google Play --> [F-droid](https://f-droid.org/en/)
- Youtube --> [Hooktube](https://hooktube.com/)
